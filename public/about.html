About Me

I am a second year engineering student at Ain Shams University, majoring in Computer Engineering and Software Systems. I am currently looking for an internship in the field of cybersecurity, data science, or software engineering.

Some other projects that I developed are:

  • 3-bit signed binary calculator:
    • Designed and implemented an arithmetic calculator that performed addition, subtraction and multiplication to two 3-bits signed binary inputs.
    • The software implemetation was made using Logisim.
    • The hardware implementation was acheived using logic gates, multiplexers, decoders and seven segment displays.
  • Synchronous counter:
    • Designed a synchronous counter to display a certain specific sequence.
    • The Hardware used in this project were JK flip flops and NE555 IC to simulate the clock signal.
    • Added a variable resistor to allow the user to control the speed of the clock signal.
    • Added the choice of manual control for the user to flip through the sequence with the push of a button.
  • A C++ logic gates solver:
    • Developed a project that processed logic gates and their inputs, checked for errors and variables, and relayed the results to the user.
    • The code could process multiple variables, request their values from the user then solve the gates.
    • In case of error, the code could also discover which type of error it was and output the problem to the user, whether it was an invalid gate or inputs.
